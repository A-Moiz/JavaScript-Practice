# JavaScript-Practice

This repository contains a collection of JavaScript practice problems used to improve problem-solving and coding skills.

Each directory represents a different problem or exercise. Inside each folder you may find:
- The problem description
- My solution
- Any additional notes or alternative approaches

## Structure

Each problem is organised into its own directory:
```
/problem-name
├── README.md (problem description)
└── solution.js
```

## Goals

- Strengthen JavaScript fundamentals
- Practice problem-solving techniques
- Explore different approaches to common coding challenges
- Track learning progress over time

## Notes

This repository is primarily for personal learning and experimentation, so solutions may evolve or be refactored over time.
